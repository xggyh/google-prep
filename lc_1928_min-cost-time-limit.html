LC 1928. Minimum Cost to Reach Destination in Time 规定时间内到达终点的最小花费

类别：🗺️ 图 / BFS / DFS · 难度：Hard · 标记：🔄 LC 变体

🗺️ 图 / BFS / DFS Hard 🔄 LC 变体

题目概览

🏷️ 类别：图 / Dijkstra 变体 | 难度：Hard | 标记：🔄

题目

EN: Graph with n cities. edges[i] = (u, v, time) is bidirectional. passingFees[i] is cost of passing through city i. Reach city n-1 from 0 within maxTime. Return min total fee.

中文：n 个城市，无向带时间边。每个城市有过路费。从 0 出发到 n-1，总时间 ≤ maxTime。求最小总过路费（含起点和终点）。

澄清问题

  • EN: Multiple edges between same pair? / Always reachable within time?
  • 中：同一对城市可能有多条不同时间的边吗？保证 maxTime 内可达吗？

🧠 整体思路

双维度 Dijkstra：传统 Dijkstra 优化"距离"；这里同时有时间和费用两个维度。

关键状态：min_cost[city] 不够 — 因为可能存在"费用高但时间短"的路径更有价值。但只要时间没超 maxTime，费用最小就是最优。

修改 Dijkstra：堆按 (cost, time, city) 排序，pop 出来的费用最小。维护 min_time[city] — 只松弛能减小时间的状态（费用已经是最小）。

为什么对：堆按 cost 出，所以第一个到达 n-1 的状态就是最小 cost。但同一个 city 可能多次 push（不同 time），所以用 min_time 剪枝。

🪜 思路拆解

Step 1 · 建图 - 无向，每条边加两次。

Step 2 · 优先队列按 cost 排序 - 元素 (cost, time, city)。 - 起点：(passingFees[0], 0, 0)。

Step 3 · 维护每个城市的最短 time - min_time[i] 初始 inf；起点 0。

Step 4 · 主循环 - pop (cost, time, u)。 - u == n-1 → return cost（最先到的就是最小）。 - 遍历邻居 (v, t)： - new_time = time + t，超 maxTime 跳过。 - new_cost = cost + passingFees[v]。 - new_time < min_time[v] 才 push（剪枝）。

⌨️ 逐步实现

Step 1-2 · 建图 + 初始化

import heapq
from collections import defaultdict

def minCost(maxTime, edges, passingFees, n=None):
    n = n or len(passingFees)
    graph = defaultdict(list)
    for u, v, t in edges:
        graph[u].append((v, t))
        graph[v].append((u, t))
    min_time = [float('inf')] * n
    min_time[0] = 0
    pq = [(passingFees[0], 0, 0)]              # (cost, time, city)

Step 4 · Dijkstra 变体

    while pq:
        cost, time, u = heapq.heappop(pq)
        if u == n - 1:
            return cost
        for v, t in graph[u]:
            new_time = time + t
            if new_time > maxTime:
                continue
            if new_time < min_time[v]:
                min_time[v] = new_time
                heapq.heappush(pq, (cost + passingFees[v], new_time, v))
    return -1

Python 完整实现

import heapq
from collections import defaultdict

def minCost(maxTime, edges, passingFees):
    n = len(passingFees)
    graph = defaultdict(list)
    for u, v, t in edges:
        graph[u].append((v, t))
        graph[v].append((u, t))
    min_time = [float('inf')] * n
    min_time[0] = 0
    pq = [(passingFees[0], 0, 0)]
    while pq:
        cost, time, u = heapq.heappop(pq)
        if u == n - 1:
            return cost
        for v, t in graph[u]:
            new_time = time + t
            if new_time > maxTime:
                continue
            if new_time < min_time[v]:
                min_time[v] = new_time
                heapq.heappush(pq, (cost + passingFees[v], new_time, v))
    return -1

讲解策略

  1. 关键洞察：堆按 cost 排序 → 第一次 pop 出终点就是答案。
  2. 用 min_time 剪枝避免重复访问。
  3. 复杂度 O(E log E)。
  4. 提替代方案：DP dp[t][i] 在时间 t 到达 i 的最小费用，O(maxTime · E)。

常见坑

  • 状态只用 cost 剪枝：会漏掉"cost 高但 time 短"的路径。
  • 忘记加起点的 passingFees：题目要求总费用含起点。
  • 重边：图保留所有边即可，Dijkstra 自动选最优。
