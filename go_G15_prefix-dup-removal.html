Google 原创 G15. Prefix Duplicates Removal 前缀去重

类别：🔤 字符串 / 解析 / 栈 · 难度：Hard · 标记：🆕 Google 原创

🔤 字符串 / 解析 / 栈 Hard 🆕 Google 原创

题目概览

🏷️ 类别：字符串 / 数组 / Set 操作 | 难度：Hard | 标记：Google 原创 | 来源：L3/L4 编译 #4

题目

EN: Delete elements from listB so that its first k elements share no value with listA.

中文：从 listB 中删除元素，使其前 k 个与 listA 无公共值。

🧠 整体思路

Set + 顺序遍历： - setA = set(listA)，O(1) 查询。 - 遍历 listB，跳过 in setA 的元素，收集前 k 个保留。

🪜 思路拆解

Step 1 · 把 listA 转 set Step 2 · 遍历 listB，跳过 in setA Step 3 · 保留前 k 个未跳过 + 之后所有元素

⌨️ 逐步实现

def remove_prefix_dup(listA, listB, k):
    setA = set(listA)
    result = []
    kept = 0
    for x in listB:
        if kept < k:
            if x not in setA:
                result.append(x)
                kept += 1
            # else skip
        else:
            result.append(x)
    return result

讲解策略

  1. Set O(1) 查询。
  2. 双状态：前 k 阶段（filter）+ 之后（全收）。

常见坑

  • k 超过 listB 中"安全"元素数 → 答案可能为空。
  • 先问 "前 k 个" 是 result 的前 k 还是 listB 原顺序的前 k。
