LC 2296. Design a Text Editor 设计文本编辑器

类别：🏗️ OOP 设计 · 难度：Hard · 标记：🔄 LC 变体

🏗️ OOP 设计 Hard 🔄 LC 变体

题目概览

🏷️ 类别：OOP 设计 / 双栈 | 难度：Hard | 标记：🔄

题目

EN: Implement: - addText(text): insert text at cursor. - deleteText(k): delete k chars before cursor; return actual deleted count. - cursorLeft(k): move cursor left k; return last 10 chars before cursor. - cursorRight(k): move cursor right k; return last 10 chars before cursor.

中文：文本编辑器，光标操作 + 删除 + 插入。

澄清问题

  • EN: Initial text? (empty) / k always valid?
  • 中：初始空？k 是否一定合法？

🧠 整体思路

两个栈，光标在中间：

[a b c d] ← cursor → [e f g h]
 left[]                right[] (倒序)
  • left[-1] = 光标紧贴左侧字符。
  • right[-1] = 光标紧贴右侧字符（栈顶 = 距离光标最近）。

所有操作 O(1) 均摊。

🪜 思路拆解

Step 1 · 状态：两个 list - left, right。

Step 2 · addText(text) - left.extend(text)。

Step 3 · deleteText(k) - k = min(k, len(left))。 - del left[-k:] 或 for _ in range(k): left.pop()。 - 返回 k。

Step 4 · cursorLeft(k) - 把 left 的 k 个 char pop 到 right。 - 返回 "".join(left[-10:])。

Step 5 · cursorRight(k) - 把 right 的 k 个 char pop 到 left。 - 返回 "".join(left[-10:])。

⌨️ 逐步实现

class TextEditor:
    def __init__(self):
        self.left = []
        self.right = []

    def addText(self, text):
        self.left.extend(text)

    def deleteText(self, k):
        k = min(k, len(self.left))
        if k:
            del self.left[-k:]
        return k

    def cursorLeft(self, k):
        while k and self.left:
            self.right.append(self.left.pop())
            k -= 1
        return "".join(self.left[-10:])

    def cursorRight(self, k):
        while k and self.right:
            self.left.append(self.right.pop())
            k -= 1
        return "".join(self.left[-10:])

讲解策略

  1. 双栈模型是核心抽象。
  2. 4 个 API 都是 O(1) 摊还。
  3. dry run 一遍 add → cursorLeft → delete。
  4. 提对比：数组实现是 O(n)。

常见坑

  • k 比剩余字符多：用 min 取小。
  • left[-10:] 不够 10 个：Python 切片不报错，直接返回有的。
  • right 栈倒序：left 是从底到顶就是文本顺序，right 是从顶到底是文本顺序。
