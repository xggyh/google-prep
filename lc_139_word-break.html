LC 139. Word Break 单词拆分

类别：🔤 字符串 / 解析 / 栈 · 难度：Med · 标记：🔄 LC 变体

🔤 字符串 / 解析 / 栈 Med 🔄 LC 变体

题目概览

🏷️ 类别：字符串 / DP | 难度：Medium | 标记：🔄

题目

EN: Given a string s and a dictionary wordDict, return true if s can be segmented into a space-separated sequence of one or more dictionary words.

中文：判断字符串 s 能否拆分成字典中单词的序列（单词可复用）。

澄清问题

  • EN: Case-sensitive? / Words can repeat? / Dictionary size?
  • 中：大小写敏感？单词可以重复使用吗？字典规模？

🧠 整体思路

DP：dp[i] = s[:i] 能否被拆分。

转移：dp[i] = True 当且仅当存在 j < i 使得 dp[j] 且 s[j:i] 在字典中。

优化：把字典转 set，O(1) 查询。

复杂度：O(n² · L_avg)，n 是 s 长度，L_avg 是平均单词长度。

🪜 思路拆解

Step 1 · 字典转 set

Step 2 · dp[0] = True（空串可拆）

Step 3 · 双层循环 - 外：i 从 1 到 n。 - 内：j 从 0 到 i-1，检查 dp[j] 且 s[j:i] in dict。

Step 4 · 返回 dp[n]

⌨️ 逐步实现

def wordBreak(s, wordDict):
    word_set = set(wordDict)
    n = len(s)
    dp = [False] * (n + 1)
    dp[0] = True
    for i in range(1, n + 1):
        for j in range(i):
            if dp[j] and s[j:i] in word_set:
                dp[i] = True
                break
    return dp[n]

讲解策略

  1. 经典 DP，子问题是"前缀能否拆"。
  2. 字典转 set 是必做优化。
  3. 复杂度 O(n²)，可通过 max_word_len 剪枝到 O(n·L_max)。

常见坑

  • dp[0] = True：空串必须可拆，否则递推不动。
  • s[j:i] 切片：是 [j, i)，符合 dp[i] 的语义（前 i 个字符）。
