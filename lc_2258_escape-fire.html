LC 2258. Escape the Spreading Fire 逃离火灾

类别：🗺️ 图 / BFS / DFS · 难度：Hard · 标记：➕ 额外推荐

🗺️ 图 / BFS / DFS Hard ➕ 额外推荐

题目概览

🏷️ 类别：图 / 多源 BFS + 二分 | 难度：Hard | 标记：➕

题目

EN: Grid with 0 (grass), 1 (fire), 2 (wall). You start at (0,0), must reach (m-1, n-1). Each minute fire spreads to adjacent grass. You move first. Return max minutes you can wait before starting, or -1 (can't escape) or 10^9 (can wait forever).

中文：网格 0 草地、1 火、2 墙。你从 (0,0) 出发到 (m-1, n-1)。每分钟火向 4 邻草地蔓延，你先动。返回最多能等多少分钟再出发；逃不掉返回 -1；可以无限等返回 10⁹。

澄清问题

  • EN: Can fire and you occupy same cell at same time? (Usually only at destination is fine) / Multiple fire sources?
  • 中：火和人同时到一格算挂吗（一般终点除外）？多个火源？

🧠 整体思路

两步： 1. 多源 BFS 算每个格子的"着火时间" fire_time[i][j]。 2. 二分等待时间 t：检查"等 t 分钟后出发能否到达终点"。

check(t)：BFS 你的移动，当前格子的到达时间 arrival = t + steps，必须 arrival < fire_time 才能进；终点特殊（arrival ≤ fire_time 也行）。

为什么二分：单调性 — 如果等 t 能逃，等 t-1 一定能；等 t 不能逃，等 t+1 一定也不能。

🪜 思路拆解

Step 1 · 多源 BFS 算 fire_time - 把所有 1（火）加入 queue，多源 BFS 同步扩散。 - fire_time[i][j] = 多少分钟后这格起火（不可达就 inf）。

Step 2 · check(t) 函数 - 你从 (0,0) 出发，t 分钟后开始走，每走一格 + 1 分钟。 - BFS 你的可达格子，arrival < fire_time 才能进。 - 终点放宽：arrival ≤ fire_time（同时到达视为逃出）。

Step 3 · 二分 t - 范围 [0, m*n]。 - 上界：若 check(m*n) 还行，return 10⁹（无限等）。

⌨️ 逐步实现

Step 1 · 多源 BFS 算 fire_time

from collections import deque

def maximumMinutes(grid):
    m, n = len(grid), len(grid[0])
    INF = float('inf')
    fire_time = [[INF] * n for _ in range(m)]
    q = deque()
    for i in range(m):
        for j in range(n):
            if grid[i][j] == 1:
                fire_time[i][j] = 0
                q.append((i, j))
    while q:
        i, j = q.popleft()
        for di, dj in [(-1,0),(1,0),(0,-1),(0,1)]:
            ni, nj = i + di, j + dj
            if 0 <= ni < m and 0 <= nj < n and grid[ni][nj] == 0 and fire_time[ni][nj] == INF:
                fire_time[ni][nj] = fire_time[i][j] + 1
                q.append((ni, nj))

Step 2 · check(t)

    def can_escape(wait):
        if fire_time[0][0] <= wait:
            return False
        visited = [[False]*n for _ in range(m)]
        visited[0][0] = True
        q = deque([(0, 0, wait)])
        while q:
            i, j, t = q.popleft()
            if (i, j) == (m-1, n-1):
                return True
            for di, dj in [(-1,0),(1,0),(0,-1),(0,1)]:
                ni, nj = i + di, j + dj
                if 0 <= ni < m and 0 <= nj < n and not visited[ni][nj] and grid[ni][nj] != 2:
                    nt = t + 1
                    is_dest = (ni == m-1 and nj == n-1)
                    if (is_dest and nt <= fire_time[ni][nj]) or (not is_dest and nt < fire_time[ni][nj]):
                        visited[ni][nj] = True
                        q.append((ni, nj, nt))
        return False

Step 3 · 二分

    lo, hi, ans = 0, m * n, -1
    while lo <= hi:
        mid = (lo + hi) // 2
        if can_escape(mid):
            ans = mid
            lo = mid + 1
        else:
            hi = mid - 1
    return 10**9 if ans == m * n else ans

Python 完整实现

from collections import deque

def maximumMinutes(grid):
    m, n = len(grid), len(grid[0])
    INF = float('inf')
    fire_time = [[INF]*n for _ in range(m)]
    q = deque()
    for i in range(m):
        for j in range(n):
            if grid[i][j] == 1:
                fire_time[i][j] = 0
                q.append((i, j))
    while q:
        i, j = q.popleft()
        for di, dj in [(-1,0),(1,0),(0,-1),(0,1)]:
            ni, nj = i+di, j+dj
            if 0 <= ni < m and 0 <= nj < n and grid[ni][nj] == 0 and fire_time[ni][nj] == INF:
                fire_time[ni][nj] = fire_time[i][j] + 1
                q.append((ni, nj))

    def can_escape(wait):
        if fire_time[0][0] <= wait:
            return False
        visited = [[False]*n for _ in range(m)]
        visited[0][0] = True
        q = deque([(0, 0, wait)])
        while q:
            i, j, t = q.popleft()
            if (i, j) == (m-1, n-1):
                return True
            for di, dj in [(-1,0),(1,0),(0,-1),(0,1)]:
                ni, nj = i+di, j+dj
                if 0 <= ni < m and 0 <= nj < n and not visited[ni][nj] and grid[ni][nj] != 2:
                    nt = t + 1
                    is_dest = (ni == m-1 and nj == n-1)
                    if (is_dest and nt <= fire_time[ni][nj]) or (not is_dest and nt < fire_time[ni][nj]):
                        visited[ni][nj] = True
                        q.append((ni, nj, nt))
        return False

    lo, hi, ans = 0, m * n, -1
    while lo <= hi:
        mid = (lo + hi) // 2
        if can_escape(mid):
            ans = mid; lo = mid + 1
        else:
            hi = mid - 1
    return 10**9 if ans == m * n else ans

讲解策略

  1. 拆两步：多源 BFS + 二分 + 模拟。
  2. 强调单调性 → 二分可行。
  3. 终点特殊处理（≤ 而非 <）。
  4. 复杂度 O(mn log(mn))。

常见坑

  • 终点不放宽：会丢掉"同时到达"的可行解。
  • 多源 BFS 漏初始化：所有火源同时入队。
  • 二分上下界：上界 m*n 足够。
