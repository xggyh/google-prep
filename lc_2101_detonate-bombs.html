LC 2101. Detonate the Maximum Bombs 引爆最多的炸弹

类别：🗺️ 图 / BFS / DFS · 难度：Med · 标记：⭐ Google 直接对应

🗺️ 图 / BFS / DFS Med ⭐ Google 直接对应

题目概览

🏷️ 类别：图 / BFS | 难度：Medium | 标记：⭐

题目

EN: bombs[i] = [x, y, r]. Detonating bomb i ignites all bombs within distance r from (x, y), recursively. Return max bombs detonated by choosing one start.

中文：每个炸弹 [x, y, r]，引爆某个炸弹会引爆所有在它半径 r 内的炸弹（递归）。选一个起点求能引爆的最多数量。

澄清问题

  • EN: Are two bombs at same position allowed? / At distance exactly r — count?
  • 中：两个炸弹同位置可以吗？正好等于 r 算引爆吗？

🧠 整体思路

两个关键洞察： 1. 不是无向图：i 能引爆 j（dist ≤ r_i）不蕴含 j 能引爆 i — 半径不对称！ 2. 距离用平方：避免浮点误差。

算法：建有向图 → 枚举起点 BFS → 取最大可达数。

复杂度：O(n²) 建图 + O(n²) 每次 BFS × n 个起点 = O(n³)。n ≤ 100 完全可。

🪜 思路拆解

Step 1 · 建有向图 - 双循环 (i, j)，i ≠ j。 - 比较 dist² ≤ r_i²，是就加边 i → j。

Step 2 · 写 BFS 函数 - 从 start 出发，记录 visited 集合。 - 返回 len(visited)。

Step 3 · 枚举所有起点取最大 - max(bfs(i) for i in range(n))。

⌨️ 逐步实现

Step 1 · 建有向图

from collections import defaultdict, deque

def maximumDetonation(bombs):
    n = len(bombs)
    graph = defaultdict(list)
    for i in range(n):
        x1, y1, r1 = bombs[i]
        for j in range(n):
            if i == j:
                continue
            x2, y2, _ = bombs[j]
            if (x1 - x2) ** 2 + (y1 - y2) ** 2 <= r1 * r1:
                graph[i].append(j)

Step 2-3 · BFS + 枚举

    def bfs(start):
        visited = {start}
        q = deque([start])
        while q:
            u = q.popleft()
            for v in graph[u]:
                if v not in visited:
                    visited.add(v)
                    q.append(v)
        return len(visited)

    return max(bfs(i) for i in range(n))

Python 完整实现

from collections import defaultdict, deque

def maximumDetonation(bombs):
    n = len(bombs)
    graph = defaultdict(list)
    for i in range(n):
        x1, y1, r1 = bombs[i]
        for j in range(n):
            if i == j: continue
            x2, y2, _ = bombs[j]
            if (x1 - x2) ** 2 + (y1 - y2) ** 2 <= r1 * r1:
                graph[i].append(j)
    def bfs(start):
        visited = {start}; q = deque([start])
        while q:
            u = q.popleft()
            for v in graph[u]:
                if v not in visited:
                    visited.add(v); q.append(v)
        return len(visited)
    return max(bfs(i) for i in range(n))

讲解策略

  1. 明确两个坑：有向图 + 距离平方。
  2. 复杂度 O(n³)，n ≤ 100 没问题。
  3. 提优化：KD-tree 索引在大 n 时降建图复杂度。

常见坑

  • 当成无向图：会漏算或多算。
  • math.sqrt：浮点比较出 bug。
  • < vs <=：恰好等于 r 算不算？默认 <=，但要问。
