Google 原创 G19. Tokens & Coins Board Game 跳格收集硬币

类别：📐 DP / 数组 · 难度：Hard · 标记：🆕 Google 原创

📐 DP / 数组 Hard 🆕 Google 原创

题目概览

🏷️ 类别：DP / 数组 | 难度：Hard | 标记：Google 原创 | 来源：L3/L4 编译 #25

题目

EN: 1D board with coins at each cell. Token can only jump exactly 3 steps forward. Maximize coins collected starting from cell 0.

中文：1D 板，每格有硬币。Token 只能往前恰好跳 3 步。求从 0 出发最大收集硬币数。

🧠 整体思路

1D DP：dp[i] = nums[i] + (dp[i+3] if i+3 < n else 0)。

倒序计算或递归 memo。

复杂度：O(N)。

🪜 思路拆解

Step 1 · dp 数组长度 n Step 2 · 倒序遍历，dp[i] = nums[i] + (dp[i+3] if valid else 0) Step 3 · 返回 dp[0]

⌨️ 逐步实现

def max_coins(nums):
    n = len(nums)
    dp = [0] * n
    for i in range(n - 1, -1, -1):
        dp[i] = nums[i] + (dp[i + 3] if i + 3 < n else 0)
    return dp[0]

讲解策略

  1. 1D DP 倒序填表。
  2. O(N) 时间 O(N) 空间（也可 O(1) 滚动）。

常见坑

  • 跳到 n-1 之外不收集硬币。
  • "恰好"3 步 vs "至多"3 步：先问。
