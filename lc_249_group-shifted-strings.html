LC 249. Group Shifted Strings 移位字符串分组

类别：🔤 字符串 / 解析 / 栈 · 难度：Med · 标记：⭐ Google 直接对应

🔤 字符串 / 解析 / 栈 Med ⭐ Google 直接对应

题目概览

🏷️ 类别：字符串 / HashMap | 难度：Medium | 标记：⭐

题目

EN: Group strings that are "shift-equivalent". Two strings are equivalent if one can be obtained from the other by shifting all chars by same amount (with wrap-around, e.g., 'z' + 1 = 'a').

中文：字符串数组，把"位移相同"的分组。如 "aaa" 和 "zzz" 一组（每位移 25），"abc" 和 "bcd" 一组。

澄清问题

  • EN: Wrap around? (Yes, 'z'→'a') / Empty string? / Case-sensitive?
  • 中：环绕（z→a）？空串？大小写敏感？

🧠 整体思路

关键 = 给每个字符串算一个"shift signature"：相邻字符的差。例如： - "abc" → 差: (1, 1)（b-a=1, c-b=1） - "bcd" → 差: (1, 1) - "aaa" → 差: (0, 0) - "zzz" → 差: (0, 0)

注意环绕：差要 mod 26。

HashMap：signature → list of strings。

复杂度：O(N · L)。

🪜 思路拆解

Step 1 · 设计 signature - 对字符串 s，signature = tuple((ord(s[i+1]) - ord(s[i])) % 26 for i in range(len(s)-1))。

Step 2 · 单字符特殊情况 - 单字符的 signature 是空 tuple ()，所有单字符归一组（其实它们都"shift-equivalent"，因为没有相邻差）。

Step 3 · HashMap 分桶

Step 4 · 返回 values

⌨️ 逐步实现

from collections import defaultdict

def groupStrings(strings):
    groups = defaultdict(list)
    for s in strings:
        if len(s) <= 1:
            sig = ()
        else:
            sig = tuple((ord(s[i+1]) - ord(s[i])) % 26 for i in range(len(s) - 1))
        groups[sig].append(s)
    return list(groups.values())

讲解策略

  1. 抽象"shift 等价" = 相邻差相同。
  2. % 26 处理环绕（z→a 的情况）。
  3. tuple 可哈希做 key。
  4. 复杂度 O(N·L)。

常见坑

  • 没 mod 26：'a'-'z' = -25 不会被识别为 'b'-'a' = 1 的等价（但 -25 mod 26 = 1）。
  • 单字符：所有空 tuple 一组。
  • 比较时算每个字符的绝对偏移：错。一定是相邻差。
