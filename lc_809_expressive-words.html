LC 809. Expressive Words 情感丰富的文字

类别：🔤 字符串 / 解析 / 栈 · 难度：Med · 标记：⭐ Google 直接对应

🔤 字符串 / 解析 / 栈 Med ⭐ Google 直接对应

题目概览

🏷️ 类别：字符串 / 双指针 | 难度：Medium | 标记：⭐

题目

EN: A string s is "stretchy" if a word w can produce s by extending some groups (consecutive same chars) to length ≥ 3. Given s and a list of words, return how many words are stretchy.

中文：字符串 s 是"伸展"的，如果某个 word 可以通过把某些连续重复字符扩展到长度 ≥ 3 而变成 s。给 s 和 words 列表，求多少 word 满足。

0522 面经里的"键盘卡键"题是这道题的简化版（去掉了 "≥ 3" 限制）。

澄清问题

  • EN: Original group of length 1 in word can be extended (to ≥ 3 in s)? / What if s has shorter group than word?
  • 中：word 中长度 1 的组可以扩展到 3+ 吗？word 的某组比 s 长怎么办？

🧠 整体思路

双指针扫 word 和 s： - 对每个连续字符组，比较 word 的长度 wlen 和 s 的长度 slen： - wlen == slen：OK。 - wlen < slen and slen >= 3：OK（被扩展了）。 - 否则：fail。

复杂度：O(N · L)，N 是 word 数，L 是 s 长度。

🪜 思路拆解

Step 1 · 写辅助函数 is_stretchy(s, word) - 两个指针 i (s)、j (word)。 - 同步走，比较字符 + 组长度。

Step 2 · 主流程 - 对每个 word 调 is_stretchy(s, word)，是就 count++。

⌨️ 逐步实现

Step 1 · is_stretchy 函数

def is_stretchy(s, word):
    i = j = 0
    while i < len(s) and j < len(word):
        if s[i] != word[j]:
            return False
        # 算两边连续组长度
        ci, cj = i, j
        while ci < len(s) and s[ci] == s[i]:
            ci += 1
        while cj < len(word) and word[cj] == word[j]:
            cj += 1
        slen, wlen = ci - i, cj - j
        if slen < wlen:
            return False
        if slen != wlen and slen < 3:
            return False
        i, j = ci, cj
    return i == len(s) and j == len(word)

Step 2 · 主函数

def expressiveWords(s, words):
    return sum(1 for w in words if is_stretchy(s, w))

讲解策略

  1. 双指针 + 组长度比较，思路清晰。
  2. 关键条件：slen >= wlen and (slen == wlen or slen >= 3)。
  3. 注意末尾边界：i == len(s) AND j == len(word) 才算完整匹配。
  4. 面经 follow-up：先把 word 压缩成 signature，整体降复杂度。

常见坑

  • 少检查 "< 3"：扩展必须扩到 3+。
  • 末尾不对齐：一边走完另一边没走完 → 不算 match。
  • 字符不同直接 return：要在比组长度之前先判字符。
