LC 772. Basic Calculator III 基本计算器 III

类别：🔤 字符串 / 解析 / 栈 · 难度：Hard · 标记：🔄 LC 变体

🔤 字符串 / 解析 / 栈 Hard 🔄 LC 变体

题目概览

🏷️ 类别：字符串 / 栈 / 解析 | 难度：Hard | 标记：🔄

题目

EN: Calculator supporting +, -, *, /, (, ), spaces, non-negative integers.

中文：支持 +-*/ 和括号的计算器。

澄清问题

  • EN: Integer division — round toward zero? / Negative results allowed?
  • 中：整数除法向 0 取整还是向下取整？结果可负吗？

🧠 整体思路

递归下降 + 栈： - 内层栈处理 * 和 / 的优先级 — 维护一个 nums 栈，遇 * 或 / 立即与栈顶结合。 - 外层递归处理 (...) — 遇到 ( 递归调用，跳过对应 )。

最后栈里全是要相加减的项（- 已转成负数），求和即可。

复杂度：O(n)。

🪜 思路拆解

Step 1 · 主函数：扫描 s，维护游标 i

Step 2 · 数字读取

Step 3 · 运算符处理 - prev_op 记录"上一个运算符"，初始 +。 - 数字读完时根据 prev_op 处理： - +: push num。 - -: push -num。 - *: pop 栈顶 *= num 再 push。 - /: pop 栈顶 int(top / num) 再 push（向 0 取整）。

Step 4 · 遇 ( 递归求子表达式

Step 5 · 遇 ) 或 end-of-string → 求 stack 和返回

⌨️ 逐步实现

def calculate(s):
    def helper(i):
        stack = []
        num = 0
        prev_op = '+'
        while i < len(s):
            ch = s[i]
            if ch.isdigit():
                num = num * 10 + int(ch)
            elif ch == '(':
                num, i = helper(i + 1)
            if (not ch.isdigit() and ch != ' ') or i == len(s) - 1:
                if prev_op == '+':
                    stack.append(num)
                elif prev_op == '-':
                    stack.append(-num)
                elif prev_op == '*':
                    stack.append(stack.pop() * num)
                elif prev_op == '/':
                    stack.append(int(stack.pop() / num))
                num = 0
                prev_op = ch
                if ch == ')':
                    return sum(stack), i
            i += 1
        return sum(stack), i

    return helper(0)[0]

讲解策略

  1. 两层结构：栈处理 */，递归处理括号。
  2. 关键 idiom：用 prev_op 延迟处理（数字读完才结合）。
  3. 整数除法用 int(a / b) 而非 a // b（Python // 向下取整，int() 向 0 取整 — 符合多数语言语义）。
  4. 复杂度 O(n)。

常见坑

  • / 取整方向：-7 // 2 = -4 但题目要 -3，必须用 int(a/b)。
  • 末尾结算：到字符串末尾或遇 ')' 都要做最后一次入栈。
  • 空格：判断条件加 ch != ' ' 避免空格触发结算。
