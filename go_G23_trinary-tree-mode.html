Google 原创 G23. Trinary Tree Mode 三叉树众数

类别：🌳 树 / BST · 难度：Hard · 标记：🆕 Google 原创

🌳 树 / BST Hard 🆕 Google 原创

题目概览

🏷️ 类别：树 / BST | 难度：Hard | 标记：Google 原创 | 来源：L3/L4 编译 #17

题目

EN: Search tree where nodes can have up to 3 children. Find the most frequently occurring value.

中文：节点可有 3 个子节点的搜索树。找出现次数最多的值。

🧠 整体思路

遍历 + Counter：DFS 遍历所有节点，Counter 统计值，返回 most_common。

进阶（BST 性质 → 中序连续相同）：如果是有序的（搜索树），中序遍历后相同值连续，可以 O(1) 空间找众数（维护当前值的计数 + 全局最大）。

🪜 思路拆解

朴素：DFS + Counter。

O(1) 空间：中序遍历，维护 cur_val, cur_count, max_count, modes。

⌨️ 逐步实现

from collections import Counter

class Node:
    def __init__(self, val):
        self.val = val
        self.children = []   # up to 3

def trinary_mode(root):
    cnt = Counter()
    def dfs(node):
        if not node:
            return
        cnt[node.val] += 1
        for c in node.children:
            dfs(c)
    dfs(root)
    if not cnt:
        return None
    return cnt.most_common(1)[0][0]

讲解策略

  1. 朴素 DFS + Counter O(n) 空间。
  2. 如果有序树可 O(1) 空间（中序连续）。
  3. 三叉用 list of children，不写死 left/mid/right。

常见坑

  • 空树 → return None。
  • 多个众数：返回任意 vs 全部，先问。
