LC 208. Implement Trie (Prefix Tree) 实现 Trie

类别：🏗️ OOP 设计 · 难度：Med · 标记：➕ 额外推荐

🏗️ OOP 设计 Med ➕ 额外推荐

题目概览

🏷️ 类别：OOP 设计 / Trie | 难度：Medium | 标记：➕

题目

EN: Implement Trie with insert(word), search(word), startsWith(prefix).

中文：实现 Trie，支持插入、查找完整词、查找前缀。

澄清问题

  • EN: Lowercase letters only? / Words can be empty? / Allow duplicate inserts?
  • 中：只小写字母？空串允许吗？同词重复插入？

🧠 整体思路

Trie 节点： - children: dict[char, TrieNode] - is_end: bool（这个节点是不是某个词的结尾）

操作： - insert(word)：从 root 出发，每个字符 setdefault 创建子节点，最后一个标记 is_end。 - search(word)：走 trie，走完且最后节点 is_end == True 返回 True。 - startsWith(prefix)：走 trie，走得通就 True（不需要 is_end）。

复杂度：每个操作 O(L)，L 是词长。

🪜 思路拆解

Step 1 · TrieNode 类

Step 2 · Trie 类持 root

Step 3 · insert - 沿 word 字符走，找不到孩子就新建。 - 最后一个节点 is_end = True。

Step 4 · 抽出辅助 _find(word) 走到对应节点 - 走不通返回 None。 - 复用给 search 和 startsWith。

Step 5 · search vs startsWith 的区别 - search 还要检查 is_end。 - startsWith 只要走通就行。

⌨️ 逐步实现

class TrieNode:
    __slots__ = ("children", "is_end")
    def __init__(self):
        self.children = {}
        self.is_end = False

class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word: str) -> None:
        node = self.root
        for ch in word:
            node = node.children.setdefault(ch, TrieNode())
        node.is_end = True

    def _find(self, word: str):
        node = self.root
        for ch in word:
            if ch not in node.children:
                return None
            node = node.children[ch]
        return node

    def search(self, word: str) -> bool:
        node = self._find(word)
        return node is not None and node.is_end

    def startsWith(self, prefix: str) -> bool:
        return self._find(prefix) is not None

讲解策略

  1. 字典实现 children 简单；如果只有 26 字母可以用 [None]*26 数组省常数。
  2. _find 抽出来避免代码重复。
  3. 复杂度每个操作 O(L)。
  4. 进阶 LC 211（带通配符）、LC 212（Word Search II）。

常见坑

  • search vs startsWith 没区分：search 必须 is_end == True。
  • setdefault 用法：第二个参数是"不存在时设的值"，返回最终的值（无论新旧）。
  • 空串：search("") 应该是 False（约定），startsWith("") 是 True（每个 Trie 都包含空前缀）。
