LC 3350. Adjacent Increasing Subarrays Detection II 检测相邻递增子数组 II

类别：📐 DP / 数组 · 难度：Med · 标记：➕ 额外推荐

📐 DP / 数组 Med ➕ 额外推荐

题目概览

🏷️ 类别：数组 / DP | 难度：Medium | 标记：➕

题目

EN: Find the maximum k such that there exist two adjacent strictly increasing subarrays of length k in nums. Return that k.

中文：找最大的 k，使得 nums 中存在两个相邻的、长度都为 k 的严格递增子数组。返回 k。

澄清问题

  • EN: "Adjacent" means they share an endpoint or are consecutive (no gap)? / Strictly increasing means <?
  • 中：相邻指首尾相接（无间隔）？严格递增是 <？

🧠 整体思路

一遍扫 + 维护两个值： - cur_len：当前以 i 结尾的递增子数组长度。 - prev_len：紧邻 cur 之前那个递增子数组的长度。

每个位置 i： - 若 nums[i] > nums[i-1]：cur_len += 1。 - 否则：prev_len = cur_len; cur_len = 1（开新段，旧段成为 prev）。 - 答案：max(ans, min(prev_len, cur_len), cur_len // 2)。 - min(prev_len, cur_len)：两个相邻段都 ≥ 某个 k。 - cur_len // 2：单个段太长时，可以从中间切两半。

🪜 思路拆解

Step 1 · 初始化 cur_len = prev_len = 1，ans = 1。

Step 2 · 遍历 i 从 1 到 n-1

Step 3 · 判断递增 - 是 → cur_len += 1。 - 否 → prev_len = cur_len，cur_len = 1。

Step 4 · 每步更新 ans - ans = max(ans, min(prev_len, cur_len), cur_len // 2)。

⌨️ 逐步实现

def maxIncreasingSubarrays(nums):
    n = len(nums)
    if n < 2:
        return 0
    cur_len = prev_len = 1
    ans = 1
    for i in range(1, n):
        if nums[i] > nums[i - 1]:
            cur_len += 1
        else:
            prev_len = cur_len
            cur_len = 1
        ans = max(ans, min(prev_len, cur_len), cur_len // 2)
    return ans

讲解策略

  1. 关键洞察：两种来源 — 两个相邻段、或一段切两半。
  2. O(n) 一遍扫，O(1) 空间。
  3. 主动 dry run 一个 [1,2,3,2,3,4,1,2] 验证。

常见坑

  • 「相邻」的定义：必须无间隔（一个紧接另一个）。
  • cur_len // 2：单段 >= 2k 时也能切成两个长 k 的段。
