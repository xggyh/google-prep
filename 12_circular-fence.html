12. 环形 Paint Fence（LC 276 变体）

来源：Circular Paint Fence · LC: LC 276 变体 🟡 变体 · 难度: Hard

DP Circular 🟡 变体 Hard

题目概览

来源：Google Software Engineer Interview Circular Paint Fence Question 原描述：LC 276 Paint Fence 但是环形 — 第一根和最后一根相邻。原帖说他和 GPT/Claude/Gemini 都没解出来。

题目

EN: Paint n fence posts arranged in a circle using k colors such that no three consecutive posts have the same color. Return the number of ways. (Posts 1 and n are adjacent.)

中文：n 根栅栏围成一圈，用 k 种颜色涂色，要求不能有连续 3 根相同颜色。第一根和第 n 根相邻。求方案数。

澄清问题

  • EN: Is the rule "no two consecutive same" (LC 276) or "no three consecutive same"? / Modular arithmetic for large answer? / n and k ranges? / n=1, n=2, n=3 edge cases — what's expected?
  • 中：约束是"任意两根相邻不同"（LC 276 原版）还是"任意三根连续不能全相同"？答案要不要模 10^9+7？n、k 范围？n 极小时（1/2/3）期望是什么？

⚠️ 关键：LC 276 原题是"不能连续 3 根相同"（at most 2 consecutive same）。本题在此基础上加上"环形"。

解法核心：固定前两根，递推剩下

线性 DP：same[i] = 第 i 根和第 i-1 根同色的方案数；diff[i] = 不同色的方案数。 - same[i] = diff[i-1] （上一根必须不同色，否则连续 3 同色） - diff[i] = (same[i-1] + diff[i-1]) * (k-1)

环形要保证：post[1] 和 post[n]、post[n-1]/post[n]/post[1] 都满足约束。

枚举前两根的颜色组合，跑线性 DP，最后检查闭环。

Python 实现

def num_ways_circular(n, k):
    if n == 0:
        return 0
    if n == 1:
        return k
    if n == 2:
        return k * k  # 两根可同可不同
    if n == 3:
        return k * k * k - k  # 总数 - 全同色

    total = 0
    # 枚举 post[0] 颜色 c0、post[1] 颜色 c1
    for c0 in range(k):
        for c1 in range(k):
            # 线性 DP, end_colors[c2] = (same_with_prev, diff_with_prev) 在位置 n-1 的方案
            # 用 (prev_prev, prev) 状态
            # 起始 dp[(c0, c1)] = 1
            dp = {(c0, c1): 1}
            for _ in range(2, n):
                ndp = {}
                for (pp, p), cnt in dp.items():
                    for c in range(k):
                        if c == p == pp:
                            continue  # 连续 3 同色
                        ndp[(p, c)] = ndp.get((p, c), 0) + cnt
                dp = ndp
            # 闭环检查：post[n-1] = p, post[0] = c0, post[1] = c1
            # 约束1：post[n-1], post[0], post[1] 不能三同
            # 约束2：post[n-2], post[n-1], post[0] 不能三同
            for (pp, p), cnt in dp.items():
                # 检查 pp, p, c0
                if pp == p == c0:
                    continue
                # 检查 p, c0, c1
                if p == c0 == c1:
                    continue
                total += cnt
    return total

⚠️ 上面 O(n·k³)，n 很大时不行。若题目要求大 n，需要矩阵快速幂 / 闭式公式。

讲解策略

  1. 先承认这题难，但分两步：(a) 解 LC 276 线性版，(b) 加环形约束。
  2. 画 4 根栅栏的小例子走一遍，让面试官跟上 DP 转移。
  3. 环形通用技巧：枚举头两根（或头一根）的状态，跑线性 DP，最后人工 check 闭环。
  4. 关键 follow-up：n 很大（10^9）？→ 矩阵快速幂，O(k⁶ log n)。
  5. 诚实表态：如果时间不够写完，列出剩余思路 — 比硬写出 bug 强。
  6. 这题原帖作者没解出来：你能在 30 分钟内画清楚思路 + 写出 30-40 行代码就是 strong hire。

常见坑

  • 理解成"不能两根相邻同色"：那是另一个变体，答案是 k·(k-1)^(n-1) ± 闭环修正。
  • 环形约束只检查 1 和 n：错，要同时检查 (n-1, n, 1) 和 (n, 1, 2)。
  • 枚举头两根 O(k²) 觉得多：N 大 K 小时完全可接受。
  • 不画图直接写代码：环形 DP 不画图很容易状态搞错。
