LC 140. Word Break II 单词拆分 II

类别：🔤 字符串 / 解析 / 栈 · 难度：Hard · 标记：🔄 LC 变体

🔤 字符串 / 解析 / 栈 Hard 🔄 LC 变体

题目概览

🏷️ 类别：字符串 / Backtrack + Memoization | 难度：Hard | 标记：🔄

题目

EN: Same as LC 139, but return all possible sentences.

中文：和 LC 139 一样，但返回所有可能的拆分方式。

澄清问题

  • 同 139。
  • EN: Are duplicate sentences possible / desired? / Max output size?
  • 中：可能有重复句子吗？输出数量上界？

🧠 整体思路

Backtrack + Memoization：dfs(i) 返回 s[i:] 的所有拆分方式（list of strings）。

关键：用 memo 避免重复计算（同一个起点会被多次访问）。

复杂度：指数级（最坏 2^n 个结果），但 memo 让中间结果共享。

🪜 思路拆解

Step 1 · 字典转 set

Step 2 · 递归 dfs(start) → 所有拆分 - 终止：start == n 返回 [""]（空尾巴）。 - 遍历 end 从 start+1 到 n： - word = s[start:end] 在字典？ - 递归 tail_results = dfs(end)。 - 把 word + tail 拼起来加到 result。

Step 3 · Memoize - @lru_cache 或手写 dict。

⌨️ 逐步实现

from functools import lru_cache

def wordBreak(s, wordDict):
    word_set = set(wordDict)
    n = len(s)

    @lru_cache(maxsize=None)
    def dfs(start):
        if start == n:
            return [""]
        result = []
        for end in range(start + 1, n + 1):
            word = s[start:end]
            if word in word_set:
                for tail in dfs(end):
                    result.append(word if not tail else word + " " + tail)
        return result

    return dfs(0)

讲解策略

  1. 比 139 复杂：要枚举所有路径而非只判存在。
  2. Memoization 关键 — 同一 start 不重算。
  3. 输出最坏指数级，要主动说。

常见坑

  • 空尾巴处理：dfs(n) 返回 [""] 而不是 []，否则上层无法拼接。
  • 拼接空格：tail 为空时不加空格。
