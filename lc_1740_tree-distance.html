LC 1740. Find Distance in a Binary Tree 找出二叉树中的距离

类别：🗺️ 图 / BFS / DFS · 难度：Med · 标记：⭐ Google 直接对应

🗺️ 图 / BFS / DFS Med ⭐ Google 直接对应

题目概览

🏷️ 类别：图 / 树 + LCA | 难度：Medium | 标记：⭐

题目

EN: Given the root of a binary tree and two values p and q, return the distance (number of edges) between the nodes with those values.

中文：二叉树中给两个节点值 p 和 q，求它们之间的距离（边数）。

澄清问题

  • EN: Are values unique? / Are p and q guaranteed to exist? / p == q case?
  • 中：节点值唯一吗？p 和 q 一定存在吗？p 等于 q 怎么办？

🧠 整体思路

公式：dist(p, q) = depth(p) + depth(q) - 2 * depth(lca(p, q))。

核心 = 找 LCA（最近公共祖先）。递归套路： - 当前节点是 p 或 q → 返回当前节点。 - 左右子树各递归找。 - 都找到 → 当前节点就是 LCA。 - 只有一边找到 → 返回那一边。

找到 LCA 后，分别从 LCA 出发 BFS/DFS 计算到 p 和 q 的距离。

🪜 思路拆解

Step 1 · 找 LCA 的递归函数 - 终止：root 为 None 或 root.val 是 p/q → 返回 root。 - 左右递归。 - 都非空 → root 就是 LCA。 - 否则返回非空那个。

Step 2 · 计算从 LCA 到目标的距离 - 简单 DFS：找到目标返回当前深度，否则递归子树取非 -1 的值。

Step 3 · 主函数组合 - lca = findLCA(root, p, q) - return dist(lca, p) + dist(lca, q)

⌨️ 逐步实现

Step 1 · 找 LCA

def findDistance(root, p, q):
    def lca(node):
        if not node or node.val == p or node.val == q:
            return node
        L = lca(node.left)
        R = lca(node.right)
        if L and R:
            return node                          # 分散在两边, 当前就是 LCA
        return L or R

Step 2 · 从某点到目标的距离

    def dist(node, target):
        if not node:
            return -1
        if node.val == target:
            return 0
        L = dist(node.left, target)
        R = dist(node.right, target)
        sub = L if L != -1 else R
        return -1 if sub == -1 else sub + 1

Step 3 · 组合

    if p == q:
        return 0
    root_lca = lca(root)
    return dist(root_lca, p) + dist(root_lca, q)

Python 完整实现

def findDistance(root, p, q):
    if p == q:
        return 0
    def lca(node):
        if not node or node.val == p or node.val == q:
            return node
        L = lca(node.left); R = lca(node.right)
        if L and R:
            return node
        return L or R

    def dist(node, target):
        if not node:
            return -1
        if node.val == target:
            return 0
        L = dist(node.left, target)
        R = dist(node.right, target)
        sub = L if L != -1 else R
        return -1 if sub == -1 else sub + 1

    root_lca = lca(root)
    return dist(root_lca, p) + dist(root_lca, q)

讲解策略

  1. 关键公式：dist = depth(p) + depth(q) - 2·depth(lca)。
  2. 递归找 LCA 是经典套路，30 秒说清楚。
  3. 复杂度 O(n)，每个节点最多访问 2 次。
  4. 提进阶：多次查询可用倍增 LCA O(log n) per query。

常见坑

  • LCA 终止条件：node.val == p 时直接返回 node（不再向下找）。
  • 同一节点 p == q：早返回 0。
  • dist 函数返回 -1：要正确传播"没找到"的信号。
