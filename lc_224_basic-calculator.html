LC 224. Basic Calculator 基本计算器

类别：🔤 字符串 / 解析 / 栈 · 难度：Hard · 标记：🔄 LC 变体

🔤 字符串 / 解析 / 栈 Hard 🔄 LC 变体

题目概览

🏷️ 类别：字符串 / 栈 / 解析 | 难度：Hard | 标记：🔄

题目

EN: Implement a basic calculator for a string s containing digits, +, -, (, ), and spaces. No multiplication or division.

中文：实现基本计算器，支持 +、-、(、)、空格。整数可多位。

澄清问题

  • EN: Can numbers be negative (e.g., -3 + 2)? / Whitespace handling? / Overflow?
  • 中：可以有一元负号吗？空格怎么处理？溢出？

🧠 整体思路

栈处理括号 + 符号传递： - 维护 result（当前累计）、sign（当前数字的符号，+1 或 -1）。 - 遇到 (：把当前 result 和 sign 压栈，重置。 - 遇到 )：当前 result *= 栈顶 sign，再加上栈顶 result。 - 遇到数字：读完整数 → result += sign * num。

复杂度：O(n) 一次遍历。

🪜 思路拆解

Step 1 · 初始化 - result = 0，sign = 1，stack = []。

Step 2 · 遍历字符 - 数字：读完整数，result += sign * num。 - +：sign = 1。 - -：sign = -1。 - (：栈保存 (result, sign)，重置 result=0, sign=1。 - )：pop 栈顶 (prev_result, prev_sign)，result = prev_result + prev_sign * result。 - 空格：跳过。

Step 3 · 返回 result

⌨️ 逐步实现

def calculate(s):
    result, sign = 0, 1
    stack = []
    i, n = 0, len(s)
    while i < n:
        ch = s[i]
        if ch.isdigit():
            num = 0
            while i < n and s[i].isdigit():
                num = num * 10 + int(s[i])
                i += 1
            result += sign * num
            continue
        elif ch == '+':
            sign = 1
        elif ch == '-':
            sign = -1
        elif ch == '(':
            stack.append(result)
            stack.append(sign)
            result, sign = 0, 1
        elif ch == ')':
            result *= stack.pop()                # prev sign
            result += stack.pop()                # prev result
        i += 1
    return result

讲解策略

  1. 栈处理嵌套是经典套路。
  2. 关键：sign 跟随括号入栈出栈。
  3. 数字可能多位，要 while 内部循环读完。
  4. 复杂度 O(n)。

常见坑

  • 空格：必须 skip。
  • 一元负号：-3+2 起点 sign = -1 已经默认处理。
  • 多位数：while 读连续数字，不要只读一位。
