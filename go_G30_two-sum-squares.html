Google 原创 G30. Two Sum-of-Squares 两种方式的平方和

类别：🧮 数学 / 概率 / DS · 难度：Med · 标记：🆕 Google 原创

🧮 数学 / 概率 / DS Med 🆕 Google 原创

题目概览

🏷️ 类别：数学 / 概率 / DS | 难度：Medium | 标记：Google 原创 | 来源：L3/L4 编译 #40

题目

EN: Find all integers s < n expressible as sum of two squares in exactly two distinct ways.

中文：找所有 s < n，s 可表示为两数平方和的方式恰好为 2 种不同方式。

🧠 整体思路

枚举所有 a² + b² (a ≤ b, a² + b² < n) 并计数。

最后找计数 == 2 的 s。

复杂度：O(n)，枚举 a 到 √n。

🪜 思路拆解

Step 1 · 双层循环 a, b in [0, √n) Step 2 · 累加 count[a² + b²] Step 3 · 过滤 count == 2 的 s

⌨️ 逐步实现

from collections import Counter
import math

def two_ways_sum_squares(n):
    cnt = Counter()
    limit = int(math.isqrt(n))
    for a in range(limit + 1):
        for b in range(a, limit + 1):
            s = a * a + b * b
            if s < n:
                cnt[s] += 1
    return sorted([s for s, c in cnt.items() if c == 2])

讲解策略

  1. 暴力枚举 + Counter 即可。
  2. a ≤ b 避免重复 (3² + 4² == 4² + 3²)。
  3. 复杂度 O(n)。

常见坑

  • 是否允许 a == b (e.g., 8 = 2² + 2²)？算 1 种。
  • 0 算不算：题目说"两数"通常允许 0。
