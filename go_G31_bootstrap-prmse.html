Google 原创 G31. Bootstrap Percentage RMSE 自助法计算 pRMSE 置信区间

类别：🧮 数学 / 概率 / DS · 难度：Med · 标记：🆕 Google 原创

🧮 数学 / 概率 / DS Med 🆕 Google 原创

题目概览

🏷️ 类别：数学 / 概率 / DS | 难度：Medium | 标记：Google 原创 | 来源：L3/L4 编译 #41

题目

EN: Compute percentage RMSE (pRMSE) for predictions, then compute its confidence interval via bootstrapping.

中文：算 pRMSE = sqrt(mean(((pred-actual)/actual)²))，再用 bootstrap 算置信区间。

🧠 整体思路

两步： 1. pRMSE = sqrt(mean(((p_i - a_i) / a_i)²)) 2. Bootstrap：从样本中有放回抽 N 个，重新算 pRMSE，重复 B 次（如 1000），取 2.5% 和 97.5% 分位数作为 95% CI。

🪜 思路拆解

Step 1 · pRMSE 函数 Step 2 · Bootstrap 循环 B 次，每次抽 N 个 Step 3 · 算 percentile

⌨️ 逐步实现

import random
import math

def pRMSE(preds, actuals):
    n = len(preds)
    err_sq = sum(((p - a) / a) ** 2 for p, a in zip(preds, actuals))
    return math.sqrt(err_sq / n)

def bootstrap_ci(preds, actuals, B=1000, alpha=0.05):
    n = len(preds)
    pairs = list(zip(preds, actuals))
    bootstrap_vals = []
    for _ in range(B):
        sample = [random.choice(pairs) for _ in range(n)]
        p_, a_ = zip(*sample)
        bootstrap_vals.append(pRMSE(p_, a_))
    bootstrap_vals.sort()
    lo = bootstrap_vals[int((alpha / 2) * B)]
    hi = bootstrap_vals[int((1 - alpha / 2) * B) - 1]
    return lo, hi

讲解策略

  1. Bootstrap 是非参数 CI 估计经典方法。
  2. 有放回抽样 = 模拟新 dataset。
  3. B = 1000 通常足够。

常见坑

  • actual == 0 时 / 0 报错，要过滤或加 epsilon。
  • 浮点精度。
