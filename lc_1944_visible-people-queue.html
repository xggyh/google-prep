LC 1944. Number of Visible People in a Queue 队列中可以看到的人数

类别：📐 DP / 数组 · 难度：Hard · 标记：⭐ Google 直接对应

📐 DP / 数组 Hard ⭐ Google 直接对应

题目概览

🏷️ 类别：DP / 数组（单调栈）| 难度：Hard | 标记：⭐

题目

EN: heights array. Person at index i can see person at j (j > i) iff min(heights[i], heights[j]) > max(heights[i+1..j-1]). Return array where ans[i] = number of people person i can see.

中文：身高数组，i 能看到 j (j > i) 当且仅当 min(h[i], h[j]) > max(h[i+1..j-1])。返回每个人能看到的人数。

面经变体：放宽规则 — "如果观察者更高，中间的矮个不挡路"。

澄清问题

  • EN: All heights distinct? / Only looking right (or both directions)?
  • 中：身高都不同吗？只看右边还是双向？

🧠 整体思路

单调栈（从右往左，递减栈）： - 维护栈：从底到顶递减的身高。 - 对每个 i 从右往左： - 能看到所有栈中比自己矮的人（弹出，count++）。 - 弹完后如果栈还非空，栈顶（比自己高）也能看到（count++）但不弹。 - 自己入栈。

为什么对：栈是从 i 右边的"递减身高"，i 能看到的就是这些 + 第一个比自己高的人。

复杂度：O(n) 摊还，每个元素最多入栈/出栈一次。

🪜 思路拆解

Step 1 · 初始化 - 单调栈（递减）。 - 结果数组 ans[i] = 0。

Step 2 · 从右往左遍历 - 对每个 i： - 弹出栈顶所有比 heights[i] 矮的，count++（这些人 i 能看到，因为它们之间没人更高）。 - 弹完后若栈非空，count++ 再多看一个（第一个比自己高的）。 - 把 heights[i] 入栈。 - 记录 ans[i] = count。

⌨️ 逐步实现

def canSeePersonsCount(heights):
    n = len(heights)
    ans = [0] * n
    stack = []                             # 递减栈, 底到顶递减

    for i in range(n - 1, -1, -1):
        count = 0
        while stack and stack[-1] < heights[i]:
            stack.pop()
            count += 1                     # 这个矮个被我看到了
        if stack:
            count += 1                     # 第一个比我高的人也能看到
        ans[i] = count
        stack.append(heights[i])
    return ans

讲解策略

  1. 关键洞察：i 能看到的人 = i 右侧第一个比 i 高的人 + 一路上递减的矮个们。
  2. 单调栈是经典 O(n) 套路。
  3. 从右往左遍历更直觉。
  4. 复杂度 O(n) 摊还，每个元素入栈出栈各一次。

常见坑

  • "等于"如何处理：< 还是 <=？严格小于的话相同身高不算看到。先问。
  • 栈方向：递减栈（底→顶递减），不是递增。
  • count 加 1 的时机：弹完后栈非空才加。栈空说明右边没有更高的了。
