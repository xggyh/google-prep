LC 394. Decode String 字符串解码

类别：🔤 字符串 / 解析 / 栈 · 难度：Med · 标记：⭐ Google 直接对应

🔤 字符串 / 解析 / 栈 Med ⭐ Google 直接对应

题目概览

🏷️ 类别：字符串 / 栈 | 难度：Medium | 标记：⭐

题目

EN: Decode string like "3[a]2[bc]" → "aaabcbc". Brackets can nest: "3[a2[c]]" → "accaccacc".

中文：解码形如 3[abc] 的字符串，重复 3 次。括号可嵌套。

澄清问题

  • EN: Numbers can be multi-digit? / Only digits, letters, brackets? / Empty brackets?
  • 中：数字可多位（10[a]）？只含数字字母括号？空括号？

🧠 整体思路

双栈解析： - num_stack：保存当前层级的"重复次数"。 - str_stack：保存当前层级之前已构建的字符串。 - 遇 [：把当前 num 和 str 压栈，重置。 - 遇 ]：pop 栈，构建 prev_str + cur_str * prev_num。

复杂度：O(总字符数)。

🪜 思路拆解

Step 1 · 初始化：cur_str = ""，cur_num = 0，两个栈。

Step 2 · 遍历 - 数字：cur_num = cur_num * 10 + int(c)。 - [：push (cur_str, cur_num)，重置。 - ]：pop，cur_str = prev_str + cur_str * prev_num。 - 字母：cur_str += c。

⌨️ 逐步实现

def decodeString(s):
    num_stack, str_stack = [], []
    cur_str, cur_num = "", 0
    for c in s:
        if c.isdigit():
            cur_num = cur_num * 10 + int(c)
        elif c == '[':
            num_stack.append(cur_num)
            str_stack.append(cur_str)
            cur_num, cur_str = 0, ""
        elif c == ']':
            n = num_stack.pop()
            prev = str_stack.pop()
            cur_str = prev + cur_str * n
        else:
            cur_str += c
    return cur_str

讲解策略

  1. 双栈 idiom 处理嵌套。
  2. 多位数累加 * 10 + int(c)。
  3. 遇 ] 立即结算。

常见坑

  • 数字只读一位 → "10[a]" 出错。
  • pop 顺序错。
  • 空括号 "0[a]" 边界。
