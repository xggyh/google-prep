LC 1044. Longest Duplicate Substring 最长重复子串

类别：🔤 字符串 / 解析 / 栈 · 难度：Hard · 标记：⭐ Google 直接对应

🔤 字符串 / 解析 / 栈 Hard ⭐ Google 直接对应

题目概览

🏷️ 类别：字符串 | 难度：Hard | 标记：⭐

题目

EN: Given string s, return the longest duplicate substring. If no duplicates exist, return "".

中文：返回字符串中最长重复子串（重复指至少出现 2 次）。无重复返回 ""。

澄清问题

  • EN: Overlapping duplicates OK? / Case-sensitive? / Length range?
  • 中：重复段可以重叠吗（如 "aaaa" 的 "aa"）？大小写敏感？长度范围？

🧠 整体思路

二分长度 + Rabin-Karp 滚动哈希： - 答案长度在 [1, n-1] 单调可二分：长度 L 有重复 ⇒ 长度 L-1 也有。 - 给定长度 L，用滚动哈希 O(n) 检查是否存在两个相同的子串。

复杂度：O(n log n)。

🪜 思路拆解

Step 1 · 二分长度框架 - lo = 1, hi = n - 1。 - 找最大 L 使 check(L) 返回某个重复子串。

Step 2 · check(L) — Rabin-Karp - base = 26（字母），mod = 大质数。 - 维护 hash window 长度 L，每次滑动 (hash * base + new) % mod - old * base^L % mod。 - 用 set 存所有窗口 hash；遇到重复 hash 比较实际子串（防止冲突）。

Step 3 · 返回最长重复串

⌨️ 逐步实现

def longestDupSubstring(s):
    n = len(s)
    MOD, BASE = (1 << 61) - 1, 26

    def check(L):
        h, p = 0, pow(BASE, L, MOD)
        seen = {}
        for i, c in enumerate(s):
            h = (h * BASE + ord(c) - ord('a')) % MOD
            if i >= L:
                h = (h - (ord(s[i - L]) - ord('a')) * p) % MOD
            if i >= L - 1:
                if h in seen and s[seen[h]:seen[h] + L] == s[i - L + 1:i + 1]:
                    return i - L + 1
                seen.setdefault(h, i - L + 1)
        return -1

    lo, hi, ans_start, ans_len = 1, n - 1, 0, 0
    while lo <= hi:
        mid = (lo + hi) // 2
        idx = check(mid)
        if idx != -1:
            ans_start, ans_len = idx, mid
            lo = mid + 1
        else:
            hi = mid - 1
    return s[ans_start:ans_start + ans_len]

讲解策略

  1. 单调性论证（长度 L OK ⇒ L-1 也 OK）→ 二分。
  2. Rabin-Karp 滚动哈希 O(1) 算新 hash。
  3. 必须比较实际子串避免 hash 冲突。
  4. 复杂度 O(n log n)。

常见坑

  • Hash 冲突没处理 → 错误答案。
  • mod 太小（如 1e9+7）容易冲突；用大质数如 (1 << 61) - 1。
  • 二分边界：lo <= hi 配合 lo = mid + 1。
