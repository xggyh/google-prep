LC 952. Largest Component by Common Factor 按公因数最大连通分量

类别：🗺️ 图 / BFS / DFS · 难度：Hard · 标记：⭐ Google 直接对应

🗺️ 图 / BFS / DFS Hard ⭐ Google 直接对应

题目概览

🏷️ 类别：图 / Union-Find | 难度：Hard | 标记：⭐

题目

EN: Array of unique ints. Two ints same group if share factor > 1. Largest group size.

中文：两数有公因数 > 1 同组。求最大组。

🧠 整体思路

UF + 因数枚举： - UF 大小 = max(nums) + 1（"数"和"因数"一起入 UF）。 - 对每个 n，枚举因数 d in [2, √n]：union(n, d) 和 union(n, n/d)。 - 统计每个 n 的 root，count 取 max。

为什么快：避免两两 GCD O(N²)。复杂度 O(N · √max)。

⌨️ 逐步实现

class UF:
    def __init__(self, n):
        self.p = list(range(n))
    def find(self, x):
        while self.p[x] != x:
            self.p[x] = self.p[self.p[x]]
            x = self.p[x]
        return x
    def union(self, x, y):
        rx, ry = self.find(x), self.find(y)
        if rx != ry:
            self.p[rx] = ry

def largestComponentSize(nums):
    M = max(nums)
    uf = UF(M + 1)
    for n in nums:
        d = 2
        while d * d <= n:
            if n % d == 0:
                uf.union(n, d)
                uf.union(n, n // d)
            d += 1
    from collections import Counter
    cnt = Counter(uf.find(n) for n in nums)
    return max(cnt.values())

讲解策略

  1. 关键：用"数 + 因数"一起入 UF。
  2. √n 枚举因数。
  3. O(N · √max) vs O(N²·log) 朴素。

常见坑

  • 两两 GCD 会 TLE。
  • 因数边界 d * d <= n。
  • UF size 必须 M+1。
