Google 原创 G21. Winning Mahjong Hand 麻将胡牌判定

类别：📐 DP / 数组 · 难度：Hard · 标记：🆕 Google 原创

📐 DP / 数组 Hard 🆕 Google 原创

题目概览

🏷️ 类别：DP / 数组 / 回溯 | 难度：Hard | 标记：Google 原创 | 来源：L3/L4 编译 #28

题目

EN: 14-tile array represents Mahjong hand. Valid winning: 1 pair (2 same) + 4 triplets (3 same OR sequential).

中文：14 张牌（整数数组），胡牌 = 1 对（2 同） + 4 顺/刻（3 张连续或相同）。

🧠 整体思路

回溯 + 剪枝： - 选一个对子（枚举可能的 pair）。 - 剩 12 张分成 4 组，每组要么 3 同（刻），要么 3 顺。 - 递归尝试每种组合。

🪜 思路拆解

Step 1 · count = Counter(tiles) Step 2 · 枚举对子：每个 count[v] >= 2 的 v 尝试做对子 Step 3 · 剩下递归分 4 组 - 取最小的 v：要么 3 同（v 三张），要么 3 顺（v, v+1, v+2 各一张）。

⌨️ 逐步实现

from collections import Counter

def is_winning(tiles):
    if len(tiles) != 14:
        return False
    count = Counter(tiles)
    vals = sorted(count.keys())

    def can_form_groups(cnt):
        if all(v == 0 for v in cnt.values()):
            return True
        # 取最小值
        v = min(k for k, c in cnt.items() if c > 0)
        # 试三同
        if cnt[v] >= 3:
            cnt[v] -= 3
            if can_form_groups(cnt):
                cnt[v] += 3
                return True
            cnt[v] += 3
        # 试三顺
        if cnt[v] >= 1 and cnt.get(v + 1, 0) >= 1 and cnt.get(v + 2, 0) >= 1:
            cnt[v] -= 1; cnt[v+1] -= 1; cnt[v+2] -= 1
            if can_form_groups(cnt):
                cnt[v] += 1; cnt[v+1] += 1; cnt[v+2] += 1
                return True
            cnt[v] += 1; cnt[v+1] += 1; cnt[v+2] += 1
        return False

    for pair_val in vals:
        if count[pair_val] >= 2:
            count[pair_val] -= 2
            if can_form_groups(count):
                count[pair_val] += 2
                return True
            count[pair_val] += 2
    return False

讲解策略

  1. 拆 pair + 4 groups。
  2. 取最小值贪心：避免重复枚举。
  3. 回溯三件套：减、递归、恢复。

常见坑

  • 不能枚举所有 pair 组合（指数）→ 枚举 1 个 pair 即可。
  • 顺子的"环"问题：1-2-3 vs 8-9-1（一般麻将不允许，先问）。
