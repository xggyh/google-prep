LC 276. Paint Fence 栅栏涂色

类别：📐 DP / 数组 · 难度：Med · 标记：🔄 LC 变体

📐 DP / 数组 Med 🔄 LC 变体

题目概览

🏷️ 类别：DP | 难度：Medium | 标记：🔄

题目

EN: n posts and k colors. Paint each post such that no three consecutive posts have the same color. Return total ways.

中文：n 根栅栏，k 种颜色，相邻 3 根不能同色。求总方案数。

澄清问题

  • EN: n, k ≥ 1? / Modular result?
  • 中：n、k 范围？要不要取模？

🧠 整体思路

DP 状态： - same[i] = 前 i 根中第 i 根和第 i-1 根同色的方案数。 - diff[i] = 第 i 根和第 i-1 根不同色的方案数。

转移： - same[i] = diff[i-1] （前一根必须不同色，否则连续 3 同） - diff[i] = (same[i-1] + diff[i-1]) * (k-1) （任意一种前面 i-1 根的合法方案，第 i 根从 k-1 种不同色里选）

总数 = same[n] + diff[n]。

🪜 思路拆解

Step 1 · 处理 n=1, n=2 特殊情况 - n=1 → k。 - n=2 → k*k。

Step 2 · 初始化 - same[2] = k（前两根同色：k 种）。 - diff[2] = k * (k-1)（前两根不同色）。

Step 3 · 递推到 n - 滚动变量节省空间。

⌨️ 逐步实现

def numWays(n, k):
    if n == 0:
        return 0
    if n == 1:
        return k
    same = k                                      # i=2: 前两同
    diff = k * (k - 1)                            # i=2: 前两异
    for i in range(3, n + 1):
        new_same = diff
        new_diff = (same + diff) * (k - 1)
        same, diff = new_same, new_diff
    return same + diff

讲解策略

  1. 状态拆"同色 / 异色"是关键。
  2. 转移方程要画图证明。
  3. O(n) 时间 O(1) 空间。

常见坑

  • n=1 边界：直接返回 k，不进 loop。
  • n=2 初始化：要在 loop 前算好。
  • (k-1) 因子：异色时新颜色有 k-1 种。
