LC 1854. Maximum Population Year 人口最多的年份

类别：📚 堆 / 扫描线 · 难度：Easy · 标记：➕ 额外推荐

📚 堆 / 扫描线 Easy ➕ 额外推荐

题目概览

🏷️ 类别：扫描线 / 差分数组 | 难度：Easy | 标记：➕

题目

EN: Given logs[i] = [birth, death] of people. Population in year y = number of people alive in y (birth ≤ y < death). Return the earliest year with maximum population.

中文：每人 [出生, 死亡] 年份。某年人口 = 出生 ≤ 该年 < 死亡 的人数。返回人口最多的最早年份。

澄清问题

  • EN: Year range? (1950-2050 typically) / Tie → earliest?
  • 中：年份范围？相同最大值返回最早？

🧠 整体思路

差分数组：在 birth 处 +1，在 death 处 -1。前缀和扫一遍找最大。

扫描线视角：每个出生是 "+1 事件"，每个死亡是 "-1 事件"。

复杂度：O(年份范围)，远小于 O(n²)。

🪜 思路拆解

Step 1 · 建差分数组 - 范围 [1950, 2050]，差分数组 delta[year - 1950]。

Step 2 · 累加差分 - delta[birth] += 1，delta[death] -= 1。

Step 3 · 前缀和扫描，记录最大 - 维护 cur 和 max_pop, max_year。

⌨️ 逐步实现

def maximumPopulation(logs):
    OFFSET = 1950
    delta = [0] * 101                            # 1950..2050
    for birth, death in logs:
        delta[birth - OFFSET] += 1
        delta[death - OFFSET] -= 1
    cur = max_pop = 0
    max_year = OFFSET
    for i, d in enumerate(delta):
        cur += d
        if cur > max_pop:
            max_pop = cur
            max_year = i + OFFSET
    return max_year

讲解策略

  1. 差分数组 + 前缀和经典套路。
  2. 复杂度 O(YEAR_RANGE) — 因为范围有限。
  3. 主动给 OFFSET，让数组下标从 0 开始。

常见坑

  • death 是开区间：出生 +1、死亡 -1（不是死亡前一年）。
  • tie：第一个达到最大值的年份才是答案，用 > 不是 >=。
