LC 743. Network Delay Time 网络延迟时间

类别：🗺️ 图 / BFS / DFS · 难度：Med · 标记：🔄 LC 变体

🗺️ 图 / BFS / DFS Med 🔄 LC 变体

题目概览

🏷️ 类别：图 / Dijkstra | 难度：Medium | 标记：🔄

题目

EN: You're given a network of n nodes labeled 1..n. times[i] = (u, v, w) represents a directed edge from u to v taking w time. Send a signal from node k. Return the time it takes for all nodes to receive the signal, or -1 if unreachable.

中文：n 个节点的有向加权图，从节点 k 发信号，求所有节点收到信号的最长时间（即最远点的最短路径长度）；不可达返回 -1。

澄清问题

  • EN: Are weights non-negative? / Can there be self-loops or duplicate edges? / How big is n?
  • 中：边权非负？有自环 / 重边吗？n 的范围？

🧠 整体思路

经典 Dijkstra：从 k 出发求单源最短路径，答案 = 所有 dist[i] 的最大值。

为什么不是 BFS：边权不一定为 1，BFS 求的是"边数最少"，不是"距离最短"。

复杂度：O((V + E) log V)，min-heap 实现。

🪜 思路拆解

Step 1 · 建邻接表 - graph[u] = [(v, w), ...]。

Step 2 · 初始化 dist 数组 - dist[k] = 0，其余 inf。

Step 3 · min-heap 反复松弛 - 堆元素 (累积距离, 节点)。 - pop 出 (d, u)，若 d > dist[u] 跳过（stale entry）。 - 遍历 u 的所有邻居 (v, w)，若 d + w < dist[v] 就更新并 push。

Step 4 · 结果聚合 - 所有 dist[i] 取 max。 - 任一 = inf → 返回 -1。

⌨️ 逐步实现

Step 1-2 · 建图 + 初始化

import heapq
from collections import defaultdict

def networkDelayTime(times, n, k):
    graph = defaultdict(list)
    for u, v, w in times:
        graph[u].append((v, w))
    dist = {i: float('inf') for i in range(1, n + 1)}
    dist[k] = 0

Step 3 · Dijkstra 主循环

    pq = [(0, k)]
    while pq:
        d, u = heapq.heappop(pq)
        if d > dist[u]:
            continue                              # stale, 已被更新
        for v, w in graph[u]:
            nd = d + w
            if nd < dist[v]:
                dist[v] = nd
                heapq.heappush(pq, (nd, v))

Step 4 · 聚合答案

    ans = max(dist.values())
    return ans if ans != float('inf') else -1

Python 完整实现

import heapq
from collections import defaultdict

def networkDelayTime(times, n, k):
    graph = defaultdict(list)
    for u, v, w in times:
        graph[u].append((v, w))
    dist = {i: float('inf') for i in range(1, n + 1)}
    dist[k] = 0
    pq = [(0, k)]
    while pq:
        d, u = heapq.heappop(pq)
        if d > dist[u]:
            continue
        for v, w in graph[u]:
            nd = d + w
            if nd < dist[v]:
                dist[v] = nd
                heapq.heappush(pq, (nd, v))
    ans = max(dist.values())
    return ans if ans != float('inf') else -1

讲解策略

  1. 复述题目：单源最短路径 + 取最大值。
  2. 选 Dijkstra 而非 BFS（边权不为 1）。
  3. 提 stale entry 处理 — 这是 Dijkstra 的关键 idiom。
  4. 复杂度 O((V+E) log V)。

常见坑

  • 忘记 if d > dist[u]: continue → 性能退化甚至错误。
  • 节点编号 1-indexed：range(1, n+1)，写成 range(n) 会丢一个节点。
  • 不可达节点没处理 → 直接返回 inf 会错。
