LC 1254. Number of Closed Islands 统计封闭岛屿的数目

类别：🗺️ 图 / BFS / DFS · 难度：Med · 标记：➕ 额外推荐

🗺️ 图 / BFS / DFS Med ➕ 额外推荐

题目概览

🏷️ 类别：图 / DFS Flood Fill | 难度：Medium | 标记：➕

题目

EN: A 2D grid with 0 (land) and 1 (water). A closed island is land surrounded by water on all sides (not touching grid boundary). Return count of closed islands.

中文：二维网格，0 是陆地，1 是水。封闭岛屿 = 四面被水包围（不接触边界）。求封闭岛屿数。

澄清问题

  • EN: Diagonal adjacency? (No, only 4-directional) / Grid size?
  • 中：邻接是 4 邻还是 8 邻？网格规模？

🧠 整体思路

Flood Fill + 边界处理： 1. 先把所有"接触边界的陆地连通块"全淹掉（DFS 把它们改成水）。 2. 剩下的陆地连通块都是封闭岛屿，DFS 计数。

为什么先处理边界：避免在主循环里判断"这个岛是否接触边界" — 简化逻辑。

复杂度：O(m·n)，每个格子最多访问 2 次。

🪜 思路拆解

Step 1 · 第一遍：淹掉边界陆地 - 遍历四条边的格子，是陆地（0）的就 DFS 把它整个连通块改成水（1）。

Step 2 · 第二遍：数剩余陆地块 - 遍历内部，每碰到 0 就 DFS 淹掉它（防止重复计数）并 count++。

Step 3 · DFS 函数 - 越界 / 是水 → 返回。 - 改成水 + 递归 4 邻。

⌨️ 逐步实现

Step 3 · DFS

def closedIsland(grid):
    m, n = len(grid), len(grid[0])

    def dfs(i, j):
        if i < 0 or i >= m or j < 0 or j >= n or grid[i][j] == 1:
            return
        grid[i][j] = 1                          # 淹掉
        dfs(i+1, j); dfs(i-1, j)
        dfs(i, j+1); dfs(i, j-1)

    for i in range(m):
        dfs(i, 0); dfs(i, n - 1)
    for j in range(n):
        dfs(0, j); dfs(m - 1, j)

Step 2 · 数封闭岛

    for i in range(1, m - 1):
        for j in range(1, n - 1):
            if grid[i][j] == 0:
                dfs(i, j)
                count += 1

Python 完整实现

def closedIsland(grid):
    m, n = len(grid), len(grid[0])
    def dfs(i, j):
        if i < 0 or i >= m or j < 0 or j >= n or grid[i][j] == 1:
            return
        grid[i][j] = 1
        dfs(i+1, j); dfs(i-1, j); dfs(i, j+1); dfs(i, j-1)
    for i in range(m):
        dfs(i, 0); dfs(i, n - 1)
    for j in range(n):
        dfs(0, j); dfs(m - 1, j)
    count = 0
    for i in range(1, m - 1):
        for j in range(1, n - 1):
            if grid[i][j] == 0:
                dfs(i, j)
                count += 1
    return count

讲解策略

  1. 两阶段策略：先处理边界，再数内部。
  2. Flood fill 用 DFS 简洁。
  3. 复杂度 O(mn)。
  4. 提优化：可以原地修改还原，或用单独 visited。

常见坑

  • 直接数岛屿不考虑边界：会把开放岛也算进去。
  • 递归深度：大网格用 BFS 或 sys.setrecursionlimit。
  • 遍历范围错：第二遍只数 [1, m-1) 内部。
